## 1.5.1
**Заменён импорт на прямой:**
``` js
// Было
import AccountIdentity from '@kaspersky/icons/[esm, build/esm, ...]/24/AccountIdentity'
// Стало
import AccountIdentity from '@kaspersky/icons/24/AccountIdentity'
```
**Не поддердивается импорт из индека:**
``` js
// Больше не поддерживается
import Icons from '@kaspersky/icons'
import { Icons8 } from '@kaspersky/icons'
// Доступен только импорт с указанием размера
import { AccountIdentity } from '@kaspersky/icons/24'
import Icons24 from '@kaspersky/icons/24'
```
