# Шаблон документации компонентов

### Документация компонента должна состоять из следующих пунктов:

1. Список Defenition of Done

2. Название компонента, описание, использование, ссылка на дизайн

3. Таблица с описанием интерфейса компонента

4. Основные виды использования компонента

Также необходимо добавить `badge`-ы:

1. Указывающий на то был ли взят за основу компонента из **Ant** или был использован нативный элемент.

2. Имеется ли полноценный дизайн для компонента, если да то прикрепить к `stories` ссылку дизайн.

3. Обладает ли компонент полной поддержкой темизации.
(все использованые цвета взяты из зеркальной палитры)

## Пример

Atoms/Button - ```src/button```
