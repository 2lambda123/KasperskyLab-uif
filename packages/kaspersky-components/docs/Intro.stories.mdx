## About
@kaspersky/components - дизайн система, созданная с целью стандартизации разработки ui-компонентов внутри лаборатории.

Является частью UIF - технологической платформы для построения пользовательских интерфейсов и веб-сервисов.

Под капотом используются: `styled-components` и `antd`.
