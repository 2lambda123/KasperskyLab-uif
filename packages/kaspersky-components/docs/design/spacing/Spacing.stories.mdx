# Внешние отступы (Spacing)

Именованные вертикальные и горизонтальные отступы до компонентов и между ними. Названия и значения соотвествуют значениям из дизайн системы -
[Margins (outer)](https://www.figma.com/file/VI67fbmFGFm4fGJxnwQIpj/02-%5BB2B%5D-Hexa-UI-Core?type=design&node-id=19424-17196&mode=design&t=lDrmnVIsKJ1C1Z1V-4)

* related (8px)
* grouped (16px)
* section (24px)
* separated (32px)

## Использование
```tsx
import styled from 'styled-components'
import spacing from '@design-system/spacing'

// Как результат выполнения функции (вернет значение c px -> 8px)
const StyledGrid = styled('div')`
  display: grid;
  grid-gap: ${spacing('related')};
`

// Как свойство (вернет только значение -> 8)
const StyledButton = styled('button')`
  margin-top: ${spacing.related}px;
`
