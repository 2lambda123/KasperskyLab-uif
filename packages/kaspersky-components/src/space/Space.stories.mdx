# Space

A component for displaying an array of elements with a certain indentation in a certain direction.

## Common
Basic Component Usage Example.

The props are passed as React.Children and the direction and size props that define the direction and size of the gap between elements, respectively.

<Story

  args={{ theme: 'light', direction: 'horizontal', size: 8, align: 'center' }}
